# codex-learning

Учебный репозиторий для практики работы с Codex, HTML, CSS и JavaScript.

## Калькулятор эффективности рекламы

Проект содержит простое веб-приложение без фреймворков и внешних библиотек. Пользователь вводит рекламный расход, количество показов, кликов и заявок, а приложение рассчитывает основные рекламные показатели:

- CTR — отношение кликов к показам;
- CPC — стоимость одного клика;
- конверсию из кликов в заявки;
- CPL — стоимость одной заявки.

## Файлы проекта

- `index.html` — структура страницы и форма ввода данных.
- `style.css` — внешний вид приложения.
- `script.js` — проверка введенных значений и расчет показателей.

## Как запустить

Откройте файл `index.html` в браузере.
