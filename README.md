# codex-learning

Учебный репозиторий для практики работы с Codex, HTML, CSS и JavaScript.

## Калькулятор эффективности рекламы

Проект содержит простое веб-приложение без фреймворков и внешних библиотек. Пользователь вводит обязательные показатели рекламной кампании:

- рекламный расход;
- количество показов;
- количество кликов;
- количество заявок.

Приложение рассчитывает основные рекламные показатели:

- CTR — отношение кликов к показам;
- CPC — стоимость одного клика;
- конверсию из кликов в заявки;
- CPL — стоимость одной заявки;
- ROMI — окупаемость рекламных вложений, если указана выручка от рекламы;
- ROI — окупаемость всех учтенных вложений, если указаны выручка от рекламы и прочие затраты.

Дополнительные финансовые поля необязательны. Если выручка не заполнена, ROMI и ROI отображаются как «—». Если заполнена только выручка, рассчитывается ROMI, а ROI остается «—». Если заполнены выручка и прочие затраты, рассчитываются оба показателя.

Ниже результатов находится отдельная зона «Целевые ориентиры». В ней можно указать плановые значения CTR, CPC, конверсии, CPL, ROMI и ROI для дальнейшего сравнения с фактическими результатами. Эти поля необязательны и не очищаются кнопкой «Очистить».

## Файлы проекта

- `index.html` — структура страницы, форма ввода данных, блок результатов и зона целевых ориентиров.
- `style.css` — внешний вид приложения.
- `script.js` — проверка введенных значений, расчет показателей и подготовка структуры целевых значений.

## Как запустить

Откройте файл `index.html` в браузере.
